# SWpedia Web Client

## Requirements

- node.js v6.1.1 or later
  - use of [nvm](https://github.com/creationix/nvm) or [nvm-windows](https://github.com/coreybutler/nvm-windows) is suggested

## Install

```
npm install
```

## Run

```
npm run watch
```

Open [localhost:3000/](http://localhost:3000/).
